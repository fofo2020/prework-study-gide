HTML

  • Hypertext Markup Language
  • Gives structure to the content
  • Main elements: Meta-releated elements, content division elements, and semantic elements.

CSS

  • Cascading Style Sheets
  • Defines how content is presented to users

Git

  • Version control System to keep track of the changes
  • Commands always start with "git"
  • Great tool for colaboration

Javascript

  • Makes static HTML/CSS webpages interactive
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
